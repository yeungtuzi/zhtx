# 纵横天下II

**纵横天下II** 是一款基于 LPMUD 引擎的中文武侠 MUD 游戏世界，以金庸武侠为背景，运行于 [MudOS-NG](https://github.com/yeungtuzi/mudos-ng) 驱动之上。

> 此仓库是原始运行环境的快照，已清理所有玩家隐私数据。

## 游戏背景

纵横天下II 是早期中文 MUD 时代的作品，构建了一个宏大的武侠江湖。游戏包含丰富的门派系统、武功技能、任务历练、帮会争霸等经典 MUD 元素。

### 游戏规模

- **10,276** 个 LPC 源码文件，**145** 个头文件
- **58** 个游戏区域，覆盖中原、西域、江南、关外等地域
- **35** 个门派/职业，囊括少林、武当、华山、明教等经典武侠门派

### 主要门派

少林、武当、华山、明教、峨眉、丐帮、星宿、桃花岛、全真教、古墓派、灵鹫宫、唐门、逍遥派等

### 主要区域

京城、杭州、苏州、成都、岳阳、泉州、西藏、西夏、华山、武当山、黑木崖、桃花岛、古墓、丐帮总舵等

### 游戏系统

- 武功技能体系（内功、剑法、刀法、拳法、轻功、暗器等数百种武功）
- 门派师徒与叛师系统
- 任务系统（朝廷任务、门派任务、杀手任务等）
- 帮会系统
- 比武竞技
- 玩家婚姻与子女系统
- 离线练功
- 公共 BBS 看板

## 目录结构

```
zhtx/
├── 6688/                  # mudlib 游戏世界根目录
│   ├── adm/               # 管理系统（守护进程、配置、模拟函数）
│   ├── class/             # 门派/职业类定义
│   ├── cmds/              # 游戏命令（玩家/巫师/管理员）
│   ├── d/                 # 游戏区域（房间、NPC、物品）
│   ├── daemon/            # 守护进程子定义（技能、状态、种族）
│   ├── data/              # 游戏数据
│   │   ├── board/         # 公开 BBS 看板（历史公告和讨论）
│   │   └── npc/           # 系统 NPC 数据
│   ├── doc/               # 游戏文档和帮助文件
│   ├── feature/           # 特征/混入（共享行为模块）
│   ├── include/           # 全局头文件和常量定义
│   ├── obj/               # 标准对象模板（用户、物品、NPC）
│   ├── quest/             # 任务系统定义
│   └── std/               # 标准继承对象（房间、角色、装备）
├── config.cfg             # MudOS 驱动运行配置
└── run.sh                 # 启动脚本
```

## 快速启动

前置条件：已编译 [MudOS-NG](https://github.com/yeungtuzi/mudos-ng) 驱动。

```bash
# 编辑 config.cfg 中的路径，使其匹配你的环境
./run.sh
```

游戏默认监听端口 `6688`，使用 MUD 客户端（如 Mudlet、Tintin++ 等）连接。

第一次启动后请注册mudadm账号,做为MUD天神(admin)。

## 技术说明

- **驱动**: MudOS v22 / FluffOS 分支，需使用 MudOS-NG 编译运行
- **语言**: LPC（LPMUD 专有脚本语言）
- **编码**: 中文内容已经从GB2312转为UTF-8,输入输出也使用UTF-8编码,以满足现代操作系统和MudOS-NG的要求，文本内容为繁体/简体中文混合
- **许可**: 参见 `6688/adm/etc/license`,本项目的内容来源非常复杂,包括了东方世界和侠客行等在内的多个早期中文MUD的内容融合,也包括了“纵横天下NCIC站“和“纵横天下II”以及衍生服务累计运行若干年间各位工作组成员的贡献,因此我无法界定其许可和版权,请各位自行斟酌.其中我写的部分(标记作者为yeung)完全放弃版权,随便使用.其他部分一般按其原授权,如果相关作者有版权方面的意见请与我(dahema@me.com)联系.
